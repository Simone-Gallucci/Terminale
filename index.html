Terminale Linux - Simulatore
user@linux:~$

Comandi Disponibili

Navigazione

  • ls - Lista file e directory
  • cd - Cambia directory
  • pwd - Mostra directory corrente
  • find - Cerca file

File Management

  • mkdir - Crea directory
  • touch - Crea file
  • rm - Rimuovi file/directory
  • rmdir - Rimuovi directory vuota
  • cp - Copia file
  • mv - Sposta/rinomina

Contenuto File

  • cat - Mostra contenuto
  • echo - Stampa testo
  • grep - Cerca in file
  • head - Prime righe
  • tail - Ultime righe
  • nano - Editor di testo

Sistema

  • ps - Processi attivi
  • whoami - Utente corrente
  • date - Data e ora
  • clear - Pulisci schermo
  • help - Mostra aiuto
